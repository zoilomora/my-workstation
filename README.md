# My Workstation
Installation and customization script for my workstation.

## Usage
Execute the following command in the bash terminal:

    $ wget https://github.com/zoilomora/my-workstation/archive/main.zip
    $ unzip main.zip
    $ cd my-workstation-main/
    $ ./setup.sh
